## Description

[Nest](https://github.com/nestjs/nest) A base API with TypeScript starter repository. Provider authentication with Passport JWT. 

Use Cloud Firestore to repositoy.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
